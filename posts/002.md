## Création et publication d'un dépôt personnel

Il existe deux façons de créer un dépot git :

### Localement

Pour créer localement un dépot git, il faut créer son projet puis utiliser la commande :
``` bash
git init
```
Avec cette ligne de commande, le dossier devient un dépot git. Ensuite, il suffit de créer son premier commit tel vu précédemment :

``` bash
git add .
git commit -m "first commit"
git push
```

Pour changer la visibilté du dépot, il faut quand même aller sur la plateforme **GitHub**, aller sur son dépot, aller dans settings, défiler en bas de page jusqu'à la *Danger Zone* et changer la visibilté ici.

### Via la plateforme **GitHub**

Sur **GitHub**, il faut cliquer sur New -> nommer le dépot -> choisir sa visibilité (publique ou privée) -> cliquer sur Create Repository

Ensuite, sur sa machine local, il faut rentrer la ligne de commande suivante :

``` bash
git remote add origin <url depot>
git branch -M main
git push -u origin main
```

*git remote add origin < url depot >* permet de relier le dépot local au dépot distant, *git branch -M main* permet de créer la branche principal et *git push -u origin main* permet de relier la branche local à la branche distante.

## Cloner un dépot Git

Le clonage d'un dépot git reviens à télécharger localement un dépot qui existe déja sur **GitHub**. Ce téléchargement implique tous les fichiers, toutes les branches et tous les commits du dépot.

### Comment cloner un dépot Git

#### Via une ligne de commande

Pour cloner un dépot git, utiliser la commande suivante dans le terminal:
``` bash
git clone https://github.com/Perla-Al-Haddad/git-atelier.git
```
L'url du dépot se situe sur la dépot github du projet. Il faut cliquer sur la flèche détoulante de l'onglet *code*, comme sur l'image suivante. 

![image](/git-atelier/images/url_depot_git.png)

#### Via un éditeur de code

Les éditeurs de code tels que **Visual Studio** et **Visual Studio Code** permette de cloner un dépot git. Ces derniers vous permettent de soit saisir l'url comme dans le terminal (emplacement du dépot), soit de se connecter à son compte github (icone github) et d'ensuite choisir un dépot à cloner.
