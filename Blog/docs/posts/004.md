## Annuler et corriger des modifications

Après avoir fait un commit, il est possible de se rendre compte qu'il y a une erreur dans le code et donc de vouloir modifier un commit. Pour cela il existe deux lignes de commande différente.

### reset

la commande git reset permet de retirer un commit en le retirant aussi de l'historique de commit. Cette commande possède trois options :

``` bash
git reset --soft <commit>
```

L'option --soft permet de retirer un commit tout en conservant tous les fichiers de ce dernier dans la staging area.

``` bash
git reset --mixed <commit>
ou
git reset <commit>
```

L'option --mixed est l'option par défaut. Elle permet de retirer un commit mais ne garde aucun fichier dans la staging area

``` bash
git reset --hard <commit>
```

L'option --hard retire le commit, enlève les fichiers de la stagin area et supprime localament les fichiers du commit. Ces fichiers ne sont perdus définitivement.

### revert

La commande revert s'écrit comme suit :

``` bash
git revert <commit>
```

Contrairement au reset, la commande revert ne supprime aucun commit. Elle permet de créer un nouveau commit qui correspond au commit précedent à celui qui lui est donné en argument.  
  
Pour ces deux commandes, on peut utliser la variable HEAD pour à la place du nom du commit. La variable HEAD est un pointeur qui pointe sur le commit actuel. Pour pointer sur les commits précedent, il faut utilier **~** de la façon HEAD~commit_actuel - commit_recherché. Par exemple, pour pointer sur le dernier commit envoyé, il faut utiliser le pointeur HEAD~1.

### Quand utilier reset ou revert

Ces deux commandes ont la même utilité mais ne laisse pas les mêmes traces dans l'historique des commits. La commande reset rend invisible l'action effectué ce qui est pratique quand le développeur est sur un dépot git car il sait ce qu'il a supprimé et pourquoi. Ceprendant, si plusieurs collaborateurs sont sur un dépot git, les autres collaborateur pourrait ne pas comprendre pourquoi un commit a disparu d'un coup, alors que la commande revert permet de l'expliquer.
